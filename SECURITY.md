# Security policy

## Supported versions

This project is at version `0.0.0` (pre-release research). Only the `main`
branch is supported.

## Reporting a vulnerability

Please report suspected security issues by email to
**samuelmausberg@gmail.com**. Do not open a public GitHub issue.

You should expect an initial response within 7 days. If the report is
confirmed, a fix and disclosure timeline will be agreed before any public
discussion.
